# Анализ проекта HBFree

_Дата: 2026-04-20_

## Тип проекта

**HBFree** — Harmonic Balance симулятор нелинейных цепей (1996–2004, Gennady Serdyuk, GPL v2).  
Реализует численный анализ нелинейных схем в частотной области методом гармонического баланса.

---

## Компоненты

| Компонент | Стек | Роль |
|-----------|------|------|
| `chanes/` | Fortran 77/90 | Основной HB-симулятор |
| `spice2hbl/` | C + flex/bison | Конвертер SPICE → HBL |
| `hbw/` | C++ | Windows-обёртка |
| `hbfree_exmpls/` | — | 8 тестовых схем + эталонные результаты |
| `hbfree_doc/` | HTML/PDF | Документация |

---

## Архитектура

### chanes/ — основной симулятор

- **`main.f`** — главный движок (~1700 строк): управление итерациями, вызов решателей
- **`libelem/`** — 24 модуля элементов: диоды, линии передачи, конденсаторы и др.
- **`lineq1.f`, `lucan.f`, `luslv.f`** — LU-факторизация и решение систем уравнений
- **`zgradie.f`, `znewsol.f`** — итерационные решатели (градиент, Ньютон)
- **`harm.f`** — FFT-преобразования
- **`frequen.f`** — формирование частотной сетки
- **`former.f`** — формирование матрицы импеданса
- **`initi.f`, `decode.f`** — инициализация и декодирование входных данных

### spice2hbl/ — двухпроходной компилятор

- **Pass 1** (`shpass1.c`) — парсинг и синтаксический анализ (~1600 строк)
- **Pass 2** (`shpass2.c`) — кодогенерация HBL (~1050 строк)
- **`shlpars.l`** — лексический анализатор (flex)
- **`shpars.y`** — грамматика (bison)

---

## Входные/выходные форматы

- **Вход:** HBL (Harmonic Balance Language) — SPICE-подобный синтаксис
- **Конвертер:** SPICE → HBL через `spice2hbl`
- **Выход:** `.raw` (двоичные результаты) + текстовые логи

---

## Типы анализов

- Гармонический баланс (одно- и двухтоновое возбуждение)
- Вычисление: напряжения, токи, спектры гармоник
- Поддержка трансмиссионных линий

## Зависимости для сборки

| Инструмент | Для чего |
|------------|---------|
| `gfortran` | компиляция Fortran (`-fdefault-real-8 -fautomatic -std=legacy`) |
| `gcc` | компиляция C (spice2hbl) |
| `flex` | лексический анализатор |
| `bison` | грамматика парсера |

```bash
cd chanes && make        # собирает симулятор → ../hbl
```
